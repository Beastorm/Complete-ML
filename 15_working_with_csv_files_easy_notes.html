What is a CSV File?

CSV means Comma Separated Values. It stores tabular data as plain text. Each row is usually one record, and values are separated by commas.

name,age,city Amit,21,Delhi Riya,22,Kolkata

A related format is TSV — Tab Separated Values — where values are separated by tabs instead of commas.

The whole video focuses on pd.read_csv(), because this is the main pandas function used to load CSV/TSV files.

Loading Local and Online CSV Files

Local file

df = pd.read_csv('aug_train.csv')

From URL/server

import requests from io import StringIO url = 'https://example.com/file.csv' r = requests.get(url) df = pd.read_csv(StringIO(r.text))
If the CSV is on the web, request the content and pass it to pandas using StringIO.

Examples Explained

TSV / tab-separated file

df = pd.read_csv('movies.tsv', sep='\t')

Without sep, pandas may treat the whole line as one column.

No column names

df = pd.read_csv('movies.tsv', sep='\t', names=['sno','name','year','rating','votes','genre'])

Use names when the file has no header row.

Custom missing values

df = pd.read_csv('data.csv', na_values=['-', '?', 'not available'])

Converters

def rename_team(name): if name == 'Royal Challengers Bangalore': return 'RCB' return name df = pd.read_csv('ipl.csv', converters={'Team1': rename_team})

Handling Large CSV Files

If a file is too large for memory, you can use nrows or chunksize.

Read sample

df = pd.read_csv('big.csv', nrows=100)

Read in chunks

chunks = pd.read_csv('big.csv', chunksize=5000) for chunk in chunks: print(chunk.shape) # apply processing here
chunksize is useful when your RAM cannot load the full dataset at once.

Final Super Simple Summary

  • CSV is the most common beginner data format in ML/data science.
  • The course covers CSV, JSON/SQL, APIs, and web scraping as major data-gathering sources.
  • pd.read_csv() can load local files and online files.
  • sep handles TSV or custom separators.
  • names and header fix missing or misplaced column names.
  • usecols, nrows, skiprows, dtype, parse_dates, converters, na_values, and encoding solve common CSV problems.
  • on_bad_lines can skip rows with parsing errors.
  • chunksize helps process very large files without loading everything into RAM.

Basic Questions to Ask

QuestionPandas FunctionWhy it matters
How big is the data?df.shapeNumber of rows and columns; tells scale.
How does data look?df.head(), df.sample()Preview rows; sample avoids first-row bias.
Data types?df.info()Shows column types, non-null counts, memory usage.
Missing values?df.isnull().sum()Find columns needing imputation/removal.
Mathematical summary?df.describe()Count, mean, std, min, percentiles, max.
Duplicates?df.duplicated().sum()Duplicate rows can hurt analysis/modeling.
Correlations?df.corr()Shows relationships among numeric columns.
🔍

Important Details

🎲

Use sample()

head() shows first rows only. sample() gives random rows and can reduce wrong first impressions.

💾

Memory optimization

info() can reveal float columns that could be converted to int to save memory in large datasets.

❓

Missing values

Titanic has many missing values in Cabin and some in Age/Embarked. Decide whether to fill or remove.

🔗

Correlation

Fare has positive relation with Survived; Pclass has negative relation. PassengerId has almost no useful relation.

📈

Correlation Explained

Correlation tells how two numeric columns move together. Pearson correlation ranges from -1 to +1.

  • ✓Positive: one increases, the other tends to increase.
  • ✓Negative: one increases, the other tends to decrease.
  • ✓Near zero: little linear relationship.
  • ✓Useful for identifying important/irrelevant features.

Final Super Simple Summary

  • Understanding data comes before modeling.
  • The video starts a sequence: basic questions, univariate analysis, multivariate analysis, and pandas profiling.
  • Use df.shape to know data size.
  • Use df.head() and preferably df.sample() to preview data.
  • Use df.info() for data types, non-null counts, and memory usage.
  • Use df.isnull().sum() to find missing values.
  • Use df.describe() for mathematical summary of numeric columns.
  • Use df.duplicated().sum() to check duplicate rows.
  • Use df.corr() to study numeric relationships and potential feature usefulness.

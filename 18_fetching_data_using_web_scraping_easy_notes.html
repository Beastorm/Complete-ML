Why Web Scraping?

If a website does not provide an API and data is not directly downloadable, web scraping may be the only option.

The video scrapes company listings from AmbitionBox: company name, rating, reviews, company type, headquarter, age, employees, and other info.

Always scrape responsibly and respect website rules, robots.txt, terms, and rate limits.
🌐

Fetching HTML with requests

First, use requests to fetch the web page HTML.

import pandas as pd import requests from bs4 import BeautifulSoup url = 'https://www.ambitionbox.com/list-of-companies?page=1' headers = {'User-Agent': 'Mozilla/5.0'} page = requests.get(url, headers=headers)
  • !A normal request may return 403 Access Denied.
  • ✓Passing a User-Agent header makes the request look like it came from a browser.
  • ✓page.text contains the HTML code.
🍲

BeautifulSoup Parsing

BeautifulSoup parses HTML and helps find elements/tags.

soup = BeautifulSoup(page.text, 'html.parser') print(soup.prettify()) soup.find_all('h2') soup.find_all('p', class_='rating')
  • ✓prettify() formats HTML to understand structure.
  • ✓Use browser Inspect Element to find tags and class names.
  • ✓Use .text.strip() to remove spaces/newlines and extract clean text.
🔍

Inspect Element Strategy

The video shows right click → Inspect to find the HTML element responsible for a visible item.

  • ✓Select the company card/container.
  • ✓Find the tag/class for company name, rating, reviews, company type, HQ, age, employees.
  • ✓If many elements use the same tag, use class names to uniquely identify the desired one.
HTML/CSS knowledge helps because scraping depends on understanding page structure.
📦

Better Approach: Scrape Each Company Card

Instead of extracting all names, all ratings, all reviews separately from the whole page, first select each company container/card and then extract details inside that card.

companies = soup.find_all('div', class_='company-content-wrapper') for company in companies: name = company.find('h2').text.strip() rating = company.find('p', class_='rating').text.strip() reviews = company.find('a', class_='review-count').text.strip()
This solves confusion when many tags/classes repeat across the full page.
🐼

Create Lists, Dictionary, and DataFrame

The scraped values are stored in lists and then converted into a pandas DataFrame.

data = { 'name': names, 'rating': ratings, 'reviews': reviews, 'company_type': company_type, 'hq': hq, 'age': age, 'employees': employees } df = pd.DataFrame(data)
🔁

Loop Over Pages

AmbitionBox has many pages. The video loops over pages and appends results into one final DataFrame.

final = pd.DataFrame() for j in range(1, 11): url = f'https://www.ambitionbox.com/list-of-companies?page={j}' page = requests.get(url, headers=headers) soup = BeautifulSoup(page.text, 'html.parser') # extract company cards and create df final = final.append(df, ignore_index=True)
  • ✓The real site had hundreds of pages.
  • ✓The video demo loops fewer pages for time.
  • ✓ignore_index=True keeps the final row index clean.

Final Super Simple Summary

  • Web scraping is used when a website does not provide an API or downloadable dataset.
  • requests.get() fetches the page HTML; sometimes you need headers like User-Agent to avoid 403 errors.
  • BeautifulSoup parses HTML and helps find tags/classes.
  • Use browser Inspect Element to identify where desired data lives.
  • Extract text using .text.strip().
  • A better approach is to first select each company card/container, then extract details from inside it.
  • Store scraped values in lists, make a dictionary, and convert it into a DataFrame.
  • Loop over pages and append/concat results with ignore_index=True.

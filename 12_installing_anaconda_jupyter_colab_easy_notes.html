Anaconda Installation

Anaconda is recommended because it bundles Python and many common data-science libraries in one place.

  • ✓Download the Individual Edition from the Anaconda website.
  • ✓Select your OS such as Windows, macOS, or Linux.
  • ✓Installation is mostly Next → Next → Finish.
  • ✓Anaconda Navigator works as a control panel for tools and environments.
Anaconda is large because it comes with many preinstalled scientific/data packages.
📓

Spyder vs Jupyter Notebook

🕷️

Spyder

A Python IDE available inside Anaconda. It can run ML/Python code and is preferred by some users.

📓

Jupyter Notebook

The teacher prefers Jupyter because it mixes code, notes, outputs, plots, images, and explanations in one notebook.

Important Jupyter points

  • ✓Runs in the browser after launching from Anaconda.
  • ✓Create folders and notebooks for each topic/project.
  • ✓Use code cells for Python code and Markdown cells for notes/headings.
  • ✓Run code using Shift + Enter.
  • ✓Can include formatted text, images, links, tables, and even interactive widgets.
  • ✓You can upload datasets and download notebooks as .ipynb or Python files.
📦

Virtual Environments

A virtual environment isolates project dependencies. This prevents every project from depending on the huge Anaconda base environment.

conda create --name campusx conda activate campusx conda install jupyter pip install numpy pandas scikit-learn conda remove --name campusx --all
  • ✓You can create environments from Anaconda Navigator or Anaconda Prompt.
  • ✓Install only libraries needed for that project.
  • ✓You can install packages from the terminal or inside notebooks using commands like !pip install numpy.
🏆

Kaggle Notebooks

Kaggle lets you run notebooks online, use public datasets, write code, share notebooks, and participate in ML competitions.

  • ✓Search a dataset, open it, and click New Notebook.
  • ✓Kaggle gives dataset paths directly in the notebook.
  • ✓You can use code and markdown just like Jupyter.
  • ✓You can save, publish, share, and download your work.
  • ✓Generated files can also be downloaded from the notebook environment.
☁️

Google Colab

Google Colab is an online Jupyter-like notebook from Google. It saves notebooks in Google Drive and is useful for ML/DL practice.

💾

Drive Save

Notebooks are saved in your Google Drive.

⚡

GPU/TPU

Runtime → Change runtime type → select GPU/TPU.

📁

Files

Upload local files or read from Drive paths.

For deep learning, Colab is useful because you can practice on GPU without buying an expensive machine.
🔗

Using Kaggle Datasets in Colab

The video shows a shortcut to avoid manually downloading a huge Kaggle dataset and then uploading it to Colab.

  1. ✓Go to Kaggle account settings and create/download the API token kaggle.json.
  2. ✓Upload kaggle.json to Colab.
  3. ✓Copy the Kaggle dataset API command from the dataset page.
  4. ✓Run the command in Colab to download the dataset directly.
  5. ✓If it downloads as zip, unzip it inside Colab.
!mkdir -p ~/.kaggle !cp kaggle.json ~/.kaggle/ !kaggle datasets download -d dataset-owner/dataset-name !unzip dataset-name.zip

Final Super Simple Summary

  • Anaconda bundles Python and data-science libraries for local ML work.
  • Jupyter Notebook lets you mix code, output, plots, markdown notes, and explanations.
  • Virtual environments isolate project dependencies and make deployment cleaner.
  • Kaggle is useful for public datasets, notebooks, sharing, and competitions.
  • Google Colab is an online notebook that saves to Drive and can use GPU/TPU.
  • For large Kaggle datasets in Colab, use the Kaggle API token instead of manual download/upload.

What is an API?

API means Application Programming Interface. In simple words, an API helps two software systems talk to each other.

Another easy meaning: APIs are data pipelines that move information from one software/system to another.
🚆

Railway booking example

IRCTC, MakeMyTrip, Yatra, etc. can show same seat availability because they communicate through APIs.

📱

Facebook example

Android, iOS, and web apps talk to the same backend/database through APIs.

Understanding JSON Response

API responses often come as JSON. A JSON viewer can help understand the structure.

  • ✓JSON looks like Python dictionaries and lists.
  • ✓TMDB response has keys such as page, results, total_pages, and total_results.
  • ✓results is a list of movie dictionaries.
  • ✓Each movie dictionary contains fields like id, title, overview, release_date, popularity, vote_average, vote_count.
💻

Python Code Flow

import pandas as pd import requests response = requests.get(url) response.status_code # 200 means success response.json()['results'] temp_df = pd.DataFrame(response.json()['results']) temp_df = temp_df[['id','title','overview','release_date','popularity','vote_average','vote_count']]
  • ✓Status code 200 means success.
  • !404 means requested resource not found; 500-type errors usually mean server-side issues.
  • ✓Convert results list into a DataFrame.
  • ✓Select only useful columns.
🔁

Pagination Loop

The first page has only some results. TMDB has many pages, so the video loops over all pages and appends each page’s DataFrame.

df = pd.DataFrame() for i in range(1, 429): response = requests.get(url.format(i)) temp_df = pd.DataFrame(response.json()['results']) temp_df = temp_df[['id','title','overview','release_date','popularity','vote_average','vote_count']] df = df.append(temp_df, ignore_index=True) df.to_csv('movies.csv')
ignore_index=True keeps the final index clean instead of restarting at 0 for every page.

Final Super Simple Summary

  • API means Application Programming Interface; it lets software systems talk.
  • Railway booking and Facebook multi-platform apps are examples of API-driven systems.
  • TMDB provides movie data through API endpoints and needs an API key.
  • API response comes in JSON with results, total_pages, and total_results.
  • Use requests.get(), response.status_code, response.json(), and pandas DataFrame conversion.
  • Loop over pages to collect all data and use ignore_index=True while appending.
  • Export the final dataset as CSV and upload to Kaggle to build your profile.
  • RapidAPI can be used to find more APIs and practice creating datasets.

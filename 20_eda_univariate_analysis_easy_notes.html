Types of Data

Before choosing plots, decide whether a column is categorical or numerical.

🏷️

Categorical

Country, gender, state, class, branch, survived, embarked, etc.

🔢

Numerical

Age, height, weight, fare, price, battery capacity, etc.

Titanic Column Understanding

  • ✓PassengerId: usually not useful for survival.
  • ✓Survived: categorical target, 0 = not survived, 1 = survived.
  • ✓Pclass: categorical passenger class 1/2/3.
  • ✓Name: high-cardinality text; needs special handling.
  • ✓Sex: categorical.
  • ✓Age and Fare: numerical.
  • ✓SibSp/Parch: numeric counts, can also be treated categorically.

Univariate Analysis for Categorical Data

For categorical columns, the main question is: how many times does each category occur?

Countplot / value_counts

sns.countplot(df['Survived']) df['Survived'].value_counts().plot(kind='bar')

Examples: Survived, Pclass, Sex, Embarked.

Pie chart

df['Survived'].value_counts().plot(kind='pie', autopct='%.2f')

Shows percentage share of each category.

  • ✓Survived count shows more people died than survived.
  • ✓Pclass count shows most passengers were in class 3.
  • ✓Sex count shows more male passengers than female.
  • ✓Embarked count shows most passengers boarded from Southampton.
🔢

📊

Histogram and Bins

A histogram divides a numerical range into intervals/bins and counts how many observations fall in each bin.

plt.hist(df['Age'], bins=20)
  • ✓More bins = more detailed but noisier view.
  • ✓Fewer bins = broader/smoother view.
  • ✓Age distribution shows fewer children and older people, more middle-aged passengers.
〰️

Distplot / KDE / PDF

Distplot is like an improved histogram with a smooth curve called KDE/PDF.

sns.distplot(df['Age'])
  • ✓PDF means Probability Density Function.
  • ✓The x-axis has values like age; y-axis shows probability density.
  • ✓Useful for understanding distribution shape and skewness.
📦

Boxplot and Five-Number Summary

Boxplot helps identify spread and outliers.

  • ✓Median = 50th percentile.
  • ✓Q1 = 25th percentile; Q3 = 75th percentile.
  • ✓IQR = Q3 - Q1.
  • ✓Lower boundary = Q1 - 1.5 × IQR.
  • ✓Upper boundary = Q3 + 1.5 × IQR.
  • ✓Values outside boundaries are potential outliers.
sns.boxplot(df['Fare']) sns.boxplot(df['Age'])
Fare has strong outliers because some passengers paid much more than most others.
📐

Skewness and Basic Stats

Distplot/histogram can show whether data is symmetric or skewed.

  • ✓Symmetric/normal: values peak near center.
  • ✓Positive/right skew: many small values, few very large values, like salary.
  • ✓Negative/left skew: many high values, few low values, like an easy exam score distribution.
df['Age'].skew() df['Age'].min() df['Age'].max() df['Age'].mean() df['Age'].std() df['Age'].var()

Final Super Simple Summary

  • Univariate analysis means analyzing one variable/column independently.
  • First identify whether the column is categorical or numerical.
  • Categorical data is analyzed using countplot/value_counts and pie charts.
  • Titanic categorical examples include Survived, Pclass, Sex, and Embarked.
  • Numerical data is analyzed using histogram, distplot/KDE, boxplot, and summary statistics.
  • Histogram shows distribution using bins; bin count affects detail.
  • Distplot/KDE/PDF shows smooth probability density and distribution shape.
  • Boxplot shows median, Q1, Q3, IQR, boundaries, and potential outliers.
  • Skewness tells whether distribution is symmetric, right-skewed, or left-skewed.

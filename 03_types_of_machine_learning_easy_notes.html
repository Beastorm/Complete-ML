2. Supervised Learning

Supervised Learning means the data has both input and output.

The model learns the relationship between input and output, then predicts the output for new input.

3. Regression vs Classification

Supervised Learning has two important types. The difference depends on the output column.

📈

Regression

Use regression when the output is a number.

  • ✓House price prediction
  • ✓Salary package prediction
  • ✓Temperature prediction
Output is numerical → Regression
🏷️

Classification

Use classification when the output is a category/class.

  • ✓Placement: Yes or No
  • ✓Email: Spam or Not Spam
  • ✓Image: Dog or Not Dog
Output is category → Classification
🔍

4. Unsupervised Learning

Unsupervised Learning means the data has only input. There is no output/label.

The model tries to discover hidden structure or patterns in the data.

🧩

5. Four Main Uses of Unsupervised Learning

🧑‍🤝‍🧑

Clustering

Group similar data points together.

Example: customer segmentation.

📉

Dimensionality Reduction

Reduce many columns into fewer useful features.

Example: PCA visualization.

🚨

Anomaly Detection

Find unusual or suspicious data points.

Example: credit-card fraud.

🛒

Association Rules

Find things that commonly happen together.

Example: bread + butter.

🌟

7. Semi-Supervised Learning

Semi-Supervised Learning is used when only a small part of the data is labeled, and a large part is unlabeled.

This is useful because labeling data is expensive and time-consuming.

Few LabelsSmall labeled data
+
Many UnlabeledLarge raw data
→
Better LearningLess manual work
Example: Google Photos groups faces. If you label one photo as “Papa,” it can label similar photos automatically.
🎮

8. Reinforcement Learning

Reinforcement Learning is different. The model does not learn from a fixed dataset.

An agent learns by interacting with an environment. Good actions get rewards; bad actions get punishments.

🤖Agent
→
🌍Environment
→
🏆Reward / Punishment
  • ✓Agent: learner or decision-maker
  • ✓Environment: world where the agent acts
  • ✓Action: what the agent does
  • ✓Reward: positive feedback
  • ✓Policy: agent’s strategy/rulebook
Examples: self-driving cars, game-playing AI, robotics, and AlphaGo by DeepMind.
🧾

Final Super Simple Summary

  • Supervised Learning: Learn from input + output.
  • Regression: Predict numbers.
  • Classification: Predict categories.
  • Unsupervised Learning: Find patterns from only input data.
  • Semi-Supervised Learning: Use few labels with lots of unlabeled data.
  • Reinforcement Learning: Learn by trial, error, reward, and punishment.
